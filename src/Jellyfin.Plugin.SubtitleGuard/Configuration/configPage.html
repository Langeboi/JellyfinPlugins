Subtitle Guard

Sync workers

Machines that do the actual subtitle work. Enroll one by running this on any Debian/Ubuntu machine that has the media mounted:

curl -sL https://raw.githubusercontent.com/Langeboi/JellyfinPlugins/main/worker/subtitle-worker/install.sh | sudo bash

Jobs are split between online workers, and idle workers automatically pull extra work from the busiest one.

Leave both fields empty if the workers mount media at the same paths as Jellyfin.
When set, the scheduled tasks only touch items under these paths. Empty = whole library. Buttons on item pages ignore this.

Whisper & oversættelse

"Generate missing subtitles" (01:00) transcribes items lacking a subtitle in the target languages - GPU workers preferred, spoken language decides the output. "Translate subtitles to Danish" (02:00) machine-translates English subs to Danish (NLLB, quality-first) for items without a Danish sub.

Items that already have a text subtitle in any of these are skipped by the transcription task.
